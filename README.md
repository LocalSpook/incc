# INCC

## Requirements

### Build requirements

- CMake >=3.28
- C++ compiler and standard library with C++23 support
- Python >=3.10
  - more_itertools package

### Development requirements

All build requirements, plus:

- Clang-Tidy >=18.1.0
- Clang-Format >=18.1.0
- Git
- EditorConfig editor plugin
